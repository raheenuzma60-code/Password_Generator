# 🔐 Password Generator

A simple command-line Password Generator built using Python. This project generates strong and random passwords based on the length specified by the user. It is designed to help users create secure passwords quickly and also demonstrates the use of Python's built-in modules for randomization.

## Features

- Generate random passwords
- User-defined password length
- Uses uppercase and lowercase letters
- Includes numbers and special characters
- Simple command-line interface
- Fast and lightweight

## Technologies Used

- Python 3
- random module
- string module

## Concepts Practiced

- User Input
- Variables
- Functions
- Loops
- Conditional Statements
- Random Number Generation
- String Manipulation

## How to Run

1. Clone this repository.
2. Open the project in Visual Studio Code.
3. Run:

```bash
python password_generator.py
```
